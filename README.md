# Salary Prediction App

A Machine Learning web application that predicts an employee's salary based on various input features such as experience, education level, job role, and other relevant factors. The application is built using **Python**, **Scikit-Learn**, and **Streamlit** to provide an interactive and user-friendly interface.

## Live Demo

 https://salary-prediction-karthik.streamlit.app/

## Overview

The Salary Prediction App helps users estimate their expected salary using a trained Machine Learning model. Users can enter their details through a simple web interface and receive instant salary predictions.

This project demonstrates the complete Machine Learning workflow:

- Data Collection
- Data Preprocessing
- Feature Engineering
- Model Training
- Model Evaluation
- Web App Deployment
